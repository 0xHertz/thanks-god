# Thanksgiving-God

## English

### Description
Thanksgiving-God is a GNOME Shell extension that solves a long-standing issue: **how to automatically switch GNOME panel font colors between light and dark modes**. This extension allows users to customize panel icon and text colors for both light and dark themes, ensuring a seamless and visually appealing experience.

### Features
- Automatically switches GNOME panel font colors based on the system's light/dark mode.
- Customizable background and foreground colors for both light and dark modes.
- Easy-to-use preferences window for color customization.
- Compatible with GNOME Shell versions 46, 47, and 48.

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/thanksgiving-god/gnome-panel-color-switcher.git
   ```
2. Copy the extension folder to your GNOME extensions directory:
   ```bash
   cp -r thanks-giving-god@gnome.com ~/.local/share/gnome-shell/extensions/
   ```
3. Restart GNOME Shell:
   - Press `Alt + F2`, type `r`, and press Enter.
4. Enable the extension using GNOME Extensions or `gnome-tweaks`.

### Usage
1. Open the preferences window via GNOME Extensions or GNOME Tweaks.
2. Customize the background and foreground colors for light and dark modes.
3. The extension will automatically apply the changes based on your system's theme.

## 中文

### 描述
Thanksgiving-God 是一个 GNOME Shell 扩展，解决了一个长期存在的问题：**如何在明亮和黑暗模式下自动切换 GNOME 面板字体颜色**。此扩展允许用户自定义明亮和黑暗主题的面板图标和文本颜色，确保流畅且视觉美观的体验。

### 功能
- 根据系统的明亮/黑暗模式自动切换 GNOME 面板字体颜色。
- 可自定义明亮和黑暗模式的背景和前景颜色。
- 简单易用的偏好设置窗口，用于颜色自定义。
- 兼容 GNOME Shell 版本 46、47 和 48。

### 安装
1. 克隆此仓库：
   ```bash
   git clone https://github.com/thanksgiving-god/gnome-panel-color-switcher.git
   ```
2. 将扩展文件夹复制到 GNOME 扩展目录：
   ```bash
   cp -r thanks-giving-god@gnome.com ~/.local/share/gnome-shell/extensions/
   ```
3. 重启 GNOME Shell：
   - 按 `Alt + F2`，输入 `r`，然后按回车。
4. 使用 GNOME Extensions 或 `gnome-tweaks` 启用扩展。

### 使用方法
1. 通过 GNOME Extensions 或 GNOME Tweaks 打开偏好设置窗口。
2. 自定义明亮和黑暗模式的背景和前景颜色。
3. 扩展将根据您的系统主题自动应用更改。
